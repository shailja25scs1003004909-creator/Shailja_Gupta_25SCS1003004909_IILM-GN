# AI-Enhanced Wildlife Corridor Identification and Leopard Movement Prediction

## Project Description
This project predicts the movement path of wildlife species using machine learning and geospatial data. The model analyzes environmental and spatial data to predict the most likely path taken by animals.

## Technologies Used
- Python
- TensorFlow / Scikit-learn
- GeoPandas
- Rasterio
- NumPy
- Pandas
- Matplotlib

## Project Workflow
1. Data Collection
2. Data Preprocessing
3. Feature Engineering
4. Model Training
5. Path Prediction
6. Visualization of Predicted Path

## How to Run
1. Install requirements:
   pip install -r requirements.txt
2. Open the notebook:
   Wildlife_Species_Path_Prediction.ipynb
3. Run all cells

## Output
The model predicts and visualizes the most probable path of the species based on input environmental conditions.
